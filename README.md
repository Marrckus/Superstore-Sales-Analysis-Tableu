Sales Trends Analysis Dashboards

This repository contains three dashboards analyzing sales performance and yearly growth trends using Python and Tableau. 
Screenshots included for quick preview.

![Dashboard 1](screenshots/Sales_Dist_by_State.png)

------------------------------------------ Insight: ------------------------------------------
Most sales come from a few top countries, which also have higher average order values and many orders. 
You can use the filter to see additional countries beyond the top 10.

![Dashboard 2](screenshots/Sales_by_Seg-Cat.png)

------------------------------------------ Insight: ------------------------------------------
Revenue is concentrated in specific categories and their key segments. 
The stacked bars highlight which products contribute most to each category, while sales, 
average order value, and order volume metrics help identify top-performing areas.

![Dashboard 3](screenshots/Sales_Trend_by_Date.png)

------------------------------------------ Insight: ------------------------------------------
The line chart shows a clear upward trend in average sales over the years, indicating steady growth. 
The heatmap visualizes revenue changes as a percentage relative to the same month of the previous year, 
highlighting periods of growth and decline across months and years.
